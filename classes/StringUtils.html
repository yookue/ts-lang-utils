Static formatPlaceholder

  • formatPlaceholder(text?, params?): undefined | null | string

  • Returns the replaced string of the source string (named placeholder) with the given parameters

    Parameters

    • Optional text: null | string

      the source string to inspect

    • Optional params: {
          [key: string]: any;
      }

      the parameters to replaced with, in the form of key values

      • [key: string]: any

    Returns undefined | null | string

    the replaced string of the source string

    Example

    StringUtils.formatPlaceholder("foo{bar}", {bar: "bar"});    //  "foobar"
    StringUtils.formatPlaceholder("foobar{none}");    //  "foobar{none}"
    StringUtils.formatPlaceholder("foobar{none}", {});    //  "foobar{none}"
